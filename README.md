# 🍳 ChefCheckApp

**Мобильное приложение для проведения чек-листов на кухне**

Приложение помогает поварам, шеф-поварам и сотрудникам контроля качества проверять соблюдение санитарных норм, сроков годности и чистоты на кухне. Удобный интерфейс и экспорт отчётов в PDF.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📋 **Чек-лист** | 24 пункта проверки, сгруппированных по разделам (общее состояние, гигиена, хранение, качество, инвентарь) |
| ✅ **Отметка статуса** | Три состояния: выполнено (+), нарушение (-), не проверено (○) |
| 📊 **Статистика** | Автоматический подсчёт выполненных пунктов и процента готовности |
| 📄 **Экспорт в PDF** | Генерация детального отчёта с таблицей всех пунктов |
| 📜 **История проверок** | Сохранение всех проверок с возможностью просмотра деталей |
| 🌙 **Тёмная тема** | Поддержка светлой и тёмной темы оформления |
| ⌨️ **Горячие клавиши** | Ctrl+S — сохранить отчёт, Ctrl+R — сбросить, Ctrl+P — PDF |

---

## 🛠 Технологии

- **.NET MAUI** 9.0 — кроссплатформенный фреймворк
- **SQLite** — локальное хранение данных
- **WebView** + HTML/JS — динамический интерфейс чек-листа
- **C#** — бэкенд и бизнес-логика
- **JavaScript** — интерактивность веб-части
